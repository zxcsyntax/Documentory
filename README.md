# ZXc Documentary

> Welcome to the official documentation for the **ZXc Project**. This guide provides everything you need to get started, from installation to advanced usage.

---

## Installation

Choose the installation method that matches your operating system below.

### Windows Install
The easiest way to get started on Windows is using our standalone installer.
* **Download:** [Windows-ZXc-V1.1 (.exe)]()

### Linux / WSL Install
To install on Linux or through Windows Subsystem for Linux, use the following commands:

```bash
# Update your repository and install the package
sudo apt update
sudo apt install zxc-documentary
```
